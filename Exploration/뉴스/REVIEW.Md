# PRT(PeerReviewTemplate)
각 항목을 스스로 확인하고 토의하여 작성한 코드에 적용합니다.
- [ ] 1.코드가 정상적으로 동작하고 주어진 문제를 해결했나요?
- [ ] 2.주석을 보고 작성자의 코드가 이해되었나요?
  > 각 코드에 대한 기능 설명
- [ ] 3.코드가 에러를 유발할 가능성이 있나요?
  > 오류 가능성 없음, 정상작동
- [ ] 4.코드 작성자가 코드를 제대로 이해하고 작성했나요?
  > 자세한 주석과 설명
- [ ] 5.코드가 간결한가요?
  > 불필요한 코드 없습니다.
